# Desktop Website for Google Search

Replace the [google](https://www.google.com/) search result from mobile to desktop website.

## Features

- Supported websites:
  - [Wikipedia](https://www.wikipedia.org/)
  - [momo購物網](https://www.momoshop.com.tw)

## Install
